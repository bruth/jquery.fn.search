Get Started

Dependencies

  • Django 1.3+
  • ModelTree
  • ordereddict (Python 2.6 only)

It is assumed Python and setuptools are already installed. If not, follow the previous two links for download and installation instructions.

Download & Install

Stable releases of Avocado are available on PyPI. Install using pip or easy_install. All dependencies are installed automatically.

Pip

pip install avocado

easy_install

easy_install avocado

Configure

Since this is about metadata managment, it is highly recommended to have the Django Admin app installed. Add avocado to your project setting's INSTALLED_APPS:

INSTALLED_APPS = (
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.messages',
    'django.contrib.sessions',

    'avocado',
    ...
)
